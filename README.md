# Muerte's Simple Knowledge

A streamlined knowledge encounter framework for Pathfinder 2e in Foundry VTT.

## Highlights
- Organize content into tabs, knowledge entries, and checks
- Post player-safe encounter cards to chat
- Use per-tab, per-entry, and per-check visibility controls
- Share structured results with optional GM-only notes
- Import and export tabs or full data sets as JSON

## Release Notes
This package is the 1.0.0 release. It applies tab DC changes as soon as the value is changed, and encounter chat cards now send a GM-visible version with DCs whenever player-facing cards hide them. This build is also prepared for GitHub/Foundry distribution with release and manifest URLs.
